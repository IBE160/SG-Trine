✨ UX Design Direction Showcase ✨

These mockups illustrate three distinct design directions for the app's key screens, using our chosen **"Refined Focus"** color theme in **light mode**. The purpose is to explore different layouts and interaction patterns before diving into detailed design.

Direction 1: The Focused List

This direction focuses on a clean, linear, and minimalist list experience. It's familiar and straightforward, presenting tasks in a simple, scrollable feed. Sub-tasks are clearly nested and indented under parent tasks. This feels like an enhanced version of a traditional to-do list, optimized for clarity and reduced cognitive load.

Direction 2: The Zen Kanban

This direction visualizes tasks as cards flowing through columns (e.g., Today, In Progress, Done). It provides a strong visual sense of progress and allows for easy drag-and-drop interaction. Sub-tasks are represented as smaller items within a card or as a checklist on the card itself, providing a structured, visual workflow.

Direction 3: The Immersive Task

This is an opinionated direction that strongly emphasizes single-task focus. The primary view is almost always a "Focus Mode," showing only the most important task or sub-task to be done NEXT. The full task list is de-emphasized and accessible only through a deliberate action, drastically reducing distractions for users prone to overwhelm.
